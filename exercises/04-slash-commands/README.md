# Exercise 04 — Slash Commands ⚡

**Goal:** Get comfortable using Copilot's built-in slash commands.

---

## What you'll learn

By the end of this exercise, you'll know how to:
- Use slash commands to trigger special features
- Switch AI models with `/model`
- Review conversation history and usage

---

## What are slash commands?

Slash commands are shortcuts that start with `/`. Instead of explaining what you
want in a full sentence, you type a short command and Copilot knows exactly what to do.

Think of them like menu buttons inside Copilot.

---

## Try these one at a time

### `/help` — See all available commands
Type:
```
/help
```
Scroll through the list. Don't worry about understanding everything — just get familiar with what's there.

---

### `/model` — Switch the AI model
Type:
```
/model
```
A menu will appear letting you choose a different AI model. Try picking one and then
ask a simple question to see if you notice any difference.

---

### `/diff` — See recent file changes
If you completed Exercise 03, type:
```
/diff
```
This shows all the changes made to files. Great for checking what Copilot did.

---

### `/clear` — Start fresh
Type:
```
/clear
```
This clears the conversation history. Copilot won't remember the earlier parts of your chat.
Useful when you want to start a new topic from scratch.

---

### `/usage` — See how much you've used
Type:
```
/usage
```
This shows how many requests you've made in this session.

---

### `/version` — Check which version you have
Type:
```
/version
```
Shows the current version of Copilot CLI installed on your computer.

---

### `/feedback` — Share your thoughts
Type:
```
/feedback
```
This opens a short survey so you can tell GitHub what you think about Copilot CLI.

---

## Bonus Challenge

Try switching modes by pressing `Shift+Tab`. You'll cycle between:
- **Interactive mode** (normal chat)
- **Plan mode** (Copilot creates a plan before acting)

Press `Shift+Tab` again to switch back.

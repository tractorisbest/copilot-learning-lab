# Exercise 01 — Ask Questions 🗣️

**Goal:** Get comfortable asking Copilot anything in plain English.

---

## What you'll learn

By the end of this exercise, you'll know how to:
- Start a conversation with Copilot
- Ask questions about a file
- Ask Copilot to explain things simply

---

## Step 1 — Open Copilot CLI

1. Open PowerShell
2. Type this and press Enter:
   ```
   cd <path-to-copilot-learning-lab>
   copilot
   ```
3. You should see the Copilot prompt appear.

---

## Step 2 — Ask a simple question

Type any of these and press Enter — just like texting a friend:

- `What is a file?`
- `What is the command line?`
- `What is GitHub Copilot?`

Read the answer. If you don't understand it, ask:
- `Can you explain that more simply?`
- `Can you give me an example?`

---

## Step 3 — Ask about a file

There's a file in this folder called `about-me.txt`. Let's ask Copilot about it.

Type this:
```
@about-me.txt what is in this file?
```

Then try:
```
@about-me.txt can you summarize this in one sentence?
```

---

## Step 4 — Ask a follow-up question

After Copilot answers, try asking:
- `What do you think I could add to this file?`
- `Can you suggest a hobby for someone who likes these things?`

---

## ✅ You're done!

You've just learned how to have a real conversation with Copilot.
Move on to **[Exercise 02 — Read & Explain](../02-read-and-explain/README.md)** when you're ready!

---

> **Tip:** There are no wrong questions. Copilot won't judge you — just ask!

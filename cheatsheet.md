# 📋 Copilot CLI Cheat Sheet

A quick reference for everything you'll use in this workspace.

---

## Starting & Stopping

| Action | Command |
|--------|---------|
| Launch Copilot CLI | `copilot` |
| Exit Copilot CLI | Press `Ctrl+C` twice |
| Clear the screen | `Ctrl+L` |

---

## Talking to Copilot

Just type in plain English! Examples:

- `"What does this file do?"`
- `"Can you explain this to me simply?"`
- `"Add a new item to my to-do list"`
- `"Fix the spelling mistakes in this file"`
- `"Create a new file called notes.txt"`

---

## Mentioning Files with @

Type `@` followed by a filename to share it with Copilot:

```
@todo.txt can you add "call the doctor" to this list?
@mystery.js what does this code do?
```

---

## Slash Commands (special Copilot features)

| Command | What it does |
|---------|-------------|
| `/help` | Show all available commands |
| `/plan` | Ask Copilot to make a plan before doing something |
| `/diff` | Show what changes were made to files |
| `/model` | Switch the AI model Copilot uses |
| `/review` | Ask Copilot to review your changes |
| `/clear` | Clear the conversation history |
| `/feedback` | Submit feedback about Copilot |
| `/version` | Show the current Copilot CLI version |
| `/exit` | Exit Copilot |

---

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate through previous commands |
| `Ctrl+C` | Cancel current action or clear input |
| `Ctrl+C` ×2 | Exit Copilot |
| `Ctrl+L` | Clear the screen |
| `Shift+Tab` | Switch between Interactive and Plan modes |
| `Esc` | Cancel the current operation |

---

## Navigating in PowerShell (outside Copilot)

| Command | What it does |
|---------|-------------|
| `cd foldername` | Go into a folder |
| `cd ..` | Go up one folder level |
| `ls` or `dir` | List files in current folder |
| `pwd` | Show where you currently are |
| `copilot` | Launch Copilot CLI |

---

## Pro Tips

1. **Be specific** — "Add 3 items to my to-do list about groceries" works better than "update the list"
2. **Ask follow-up questions** — If you don't understand an answer, just ask "can you explain that differently?"
3. **Use `/plan` first for big tasks** — It helps Copilot think through the steps before acting
4. **Nothing is permanent** — You can always ask Copilot to undo something or start over

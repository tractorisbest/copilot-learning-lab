## ✅ Step 1 — Finish Creating GitHub Repo

You should be on GitHub right now at: https://github.com/new

Look for the green button at the bottom and click:
- **"Create repository"**

Wait for the page to load. You'll see a page with a green "Code" button.

---

## ✅ Step 2 — Copy Your Repository URL

On the new repository page:
1. Click the green **"Code"** button
2. Make sure **HTTPS** is selected (not SSH)
3. Copy the URL - it will look like:
   ```
   https://github.com/YOUR-USERNAME/copilot-learning-lab.git
   ```

---

## ✅ Step 3 — Paste URL Here & Run Commands

Open PowerShell and run these commands:

```powershell
cd C:\Users\ck\copilot-learning-lab

git remote add origin PASTE-YOUR-URL-HERE

git branch -M main

git push -u origin main
```

**Replace `PASTE-YOUR-URL-HERE`** with the URL you copied in Step 2.

When you press Enter on the last command, GitHub will ask for your password or personal access token. Enter it and you're done!

---

## 🎉 That's It!

Once the push finishes, your project is LIVE on GitHub!

Share this link:
```
https://github.com/YOUR-USERNAME/copilot-learning-lab
```

## ❓ Need Help?

- **Can't remember the URL?** Go back to GitHub, click Code, and copy it again
- **Authentication failed?** Use a Personal Access Token instead of password:
  https://github.com/settings/personal-access-tokens
- **Permission denied?** Make sure you're using HTTPS (not SSH)

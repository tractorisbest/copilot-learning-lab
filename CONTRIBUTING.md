# Contributing to Copilot Learning Lab

Thanks for your interest in making this learning resource better! 🙌

## Ways You Can Help

### 🐛 Found a bug or typo?
- Open an issue describing what you found
- Or fix it directly and submit a pull request

### 💡 Have an idea for a new exercise?
- Open an issue with your suggestion
- Include what the exercise would teach and why it's important
- We'll discuss it together!

### 📝 Want to improve existing content?
- Fork the repo
- Make your changes
- Submit a pull request with a clear description of what you improved

### ✨ Found a creative sandbox exercise?
- Share it! Add it to the `sandbox/` folder
- Include a README explaining what it teaches

## How to Submit Changes

1. **Fork** this repository
2. **Create a branch** for your changes: `git checkout -b improve-exercise-2`
3. **Make your changes** and test them (walk through the exercise yourself)
4. **Commit with a clear message**: `git commit -m "Clarify explanation in exercise 2"`
5. **Push** to your fork: `git push origin improve-exercise-2`
6. **Open a Pull Request** on GitHub

## Code of Conduct

- Be respectful and kind
- Welcome people of all experience levels
- Focus on helping others learn
- No spam or self-promotion

## Questions?

Open an issue or check the main [README.md](README.md) for getting started.

Thanks for making this better! 💙

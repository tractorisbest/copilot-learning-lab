## How to Use Starter Files

### Step 1 — Read the Exercise README
Each exercise folder has a `README.md` that explains what to do

### Step 2 — Use the `@` Symbol
Share the file with Copilot:
```
@filename what should I do?
```

### Step 3 — Ask in Plain English
```
@todo.txt add this item: learn programming
@mystery.js explain the first function
```

### Step 4 — Keep Asking Follow-Ups
```
Can you explain that differently?
Can you give me an example?
What does [word] mean?
```

# 🎨 Sandbox — Free Practice Area

Welcome to the sandbox! This is your playground for experimenting without any rules or pressure.

---

## What's Here

| File | What it is |
|---|---|
| `daily-growth-journal.md` | A template for daily reflection and AI learning |
| *(You can add more!)* | Create your own practice files |

---

## How to Use the Sandbox

1. **Copy and modify** — Take the journal template and make it your own
2. **Create new files** — Add your own practice exercises
3. **Experiment freely** — Ask Copilot to help you with anything
4. **Break things** — Don't worry, you can't break anything permanently!

---

## Practice Ideas

### 1. Customize the Journal
```
@sandbox/daily-growth-journal.md can you add a section for [something important to me]?
```

### 2. Create a Personal FAQ
Ask Copilot to create a file with:
- Questions you often ask
- Answers Copilot provides
- Your own notes and reflections

### 3. Build a Learning Tracker
Create a file to track:
- What you learned this week
- Projects you've completed
- Skills you want to develop

### 4. Experiment with `/plan`
Use `/plan` mode to ask Copilot to help you:
- Organize a project
- Plan your week
- Design something from scratch

### 5. Practice Different Question Styles
Try asking Copilot the same question in different ways:
- As a beginner (simple)
- As an expert (detailed)
- As a teacher (explaining to others)

Notice how the answers change!

---

## Tips for the Sandbox

- **Use `@` liberally** — share files, ask for feedback
- **Ask follow-ups** — "Can you explain that differently?"
- **Experiment with `/model`** — try different AI models and compare
- **Save interesting responses** — you might want them later
- **Share your creations** — contribute back to the project!

# 🚀 Copilot CLI Learning Lab

Welcome! This is a **free, beginner-friendly training ground** for learning the command line and GitHub Copilot CLI — no coding experience needed.

Think of this as a gym for learning. Work through the exercises at your own pace, practice in the sandbox, and become confident using AI in the terminal.

---

## What's Included

| Folder | What's Inside |
|---|---|
| **exercises/** | 5 guided step-by-step lessons (start here!) |
| **sandbox/** | Free practice area — experiment with no pressure |
| **cheatsheet.md** | Quick reference for commands you'll use often |

---

## How to Use This

1. **Open PowerShell** (Windows) or Terminal (Mac/Linux)
2. **Navigate to this folder:**
   ```
   cd <path-to-copilot-learning-lab>
   ```
3. **Launch Copilot CLI:**
   ```
   copilot
   ```
4. **You're in!** Just type in plain English. Talk to Copilot like a friend.

---

## The 5 Exercises (in order)

| # | Exercise | What you'll learn |
|---|----------|-------------------|
| 1 | [Ask Questions](exercises/01-ask-questions/README.md) | How to ask Copilot anything |
| 2 | [Read & Explain](exercises/02-read-and-explain/README.md) | How to ask Copilot to explain a file |
| 3 | [Edit Files](exercises/03-edit-files/README.md) | How to ask Copilot to change a file |
| 4 | [Slash Commands](exercises/04-slash-commands/README.md) | How to use special Copilot commands |
| 5 | [Plan & Build](exercises/05-plan-and-build/README.md) | How to build something from scratch |

Each exercise takes **10-30 minutes**. Do them in order — they build on each other!

---

## Quick Tips

- **Tag files with `@`** — type `@filename` to share a file with Copilot
- **Use `/help`** — see all available commands anytime
- **No wrong questions** — ask Copilot anything. It won't judge you!
- **Press `Ctrl+C` twice** — exit Copilot when you're done

---

## What You'll Be Able to Do

After these 5 exercises, you'll know how to:
- ✅ Ask Copilot questions in plain English
- ✅ Use `@` to share files
- ✅ Edit and create files with Copilot's help
- ✅ Use special commands (`/plan`, `/help`, `/model`, etc.)
- ✅ Plan and build projects from scratch

## System Requirements

- **Windows:** PowerShell v6 or higher
- **Mac/Linux:** Bash or Zsh terminal
- **Subscription:** Active GitHub Copilot subscription
- **CLI installed:** `copilot` command available in your terminal

Not sure if you have it? Type:
```
copilot --version
```
